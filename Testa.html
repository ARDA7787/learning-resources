              <p class="text-lg leading-relaxed mb-4">

                    <i class="fas fa-certificate text-yellow-600 mt-1 mr-3"></i>

          <h2 class="font-tiempos text-4xl font-bold mb-8 text-gray-900">2. Reconnaissance and Information Gathering</h2>


          <div class="space-y-8">

            <div>

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">2.1 Passive Reconnaissance Techniques</h3>

              <p class="text-lg leading-relaxed mb-4">

                <strong>Passive reconnaissance is the initial phase of information gathering where an attacker collects data about a target without directly interacting with its systems</strong>. The goal is to gather publicly available information to build a profile of the target and identify potential attack vectors.

              </p>


              <div class="bg-blue-50 p-6 rounded-lg border-l-4 border-blue-400 mb-6">

                <h4 class="font-semibold text-lg mb-3 text-blue-900">Key Techniques:</h4>

                <ul class="space-y-2 text-blue-800">

                  <li><strong>Search engine queries (Google dorking)</strong> - Using advanced search operators to find sensitive files</li>

                  <li><strong>WHOIS lookups</strong> - Revealing domain registration details</li>

                  <li><strong>DNS enumeration</strong> - Querying DNS records to map network infrastructure</li>

                  <li><strong>Social media intelligence</strong> - Gathering employee and organizational information</li>

                </ul>

              </div>

            </div>


            <div>

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">2.2 Active Reconnaissance and Enumeration</h3>

              <p class="text-lg leading-relaxed mb-4">

                <strong>Active reconnaissance involves directly interacting with the target&#39;s systems to gather information</strong>, which is more intrusive than passive methods but provides more detailed and accurate data.

              </p>


              <div class="code-block mb-6">

                <pre><code># Ping sweep to identify live hosts

fping -g 192.168.1.0/24 2&gt;/dev/null


# Nmap port scanning with service detection

nmap -sS -sV -O -T4 target.com


# Netcat for basic service interaction

nc -zv target.com 22

nc -zv target.com 80</code></pre>

              </div>

          <h2 class="font-tiempos text-4xl font-bold mb-8 text-gray-900">5. Malware Development for Linux (Beginner to Advanced)</h2>


          <div class="space-y-8">

            <div>

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">5.1 Fundamentals of Linux Malware</h3>

              <p class="text-lg leading-relaxed mb-4">

                <strong>Linux malware can be broadly categorized based on where it operates and the level of privileges it attains</strong>. Understanding the distinction between <strong>user-space</strong> and <strong>kernel-space</strong> is fundamental to grasping the capabilities and detection mechanisms of different types of Linux malware.

              </p>


              <div class="grid md:grid-cols-2 gap-6 mb-6">

                <div class="bg-white p-6 rounded-lg border border-gray-200">

                  <div class="flex items-center mb-4">

                    <i class="fas fa-user text-2xl text-blue-600 mr-3"></i>

                    <h4 class="font-semibold text-lg text-gray-800">User-Space Malware</h4>

                  </div>

                  <ul class="space-y-2 text-sm text-gray-700">

                    <li>• Operates with user privileges</li>

                    <li>• Easier to develop and deploy</li>

                    <li>• More likely to be detected</li>

                    <li>• Limited system access</li>

                    <li>• Examples: Shell scripts, Python backdoors</li>

                  </ul>

                </div>

                <div class="bg-white p-6 rounded-lg border border-gray-200">

                  <div class="flex items-center mb-4">

                    <i class="fas fa-cog text-2xl text-red-600 mr-3"></i>

                    <h4 class="font-semibold text-lg text-gray-800">Kernel-Space Malware</h4>

                  </div>

                  <ul class="space-y-2 text-sm text-gray-700">

                    <li>• Highest privilege level (Ring 0)</li>

                    <li>• More powerful and stealthy</li>

                    <li>• Complex development</li>

                    <li>• Direct hardware access</li>

                    <li>• Examples: Rootkits, LKMs</li>

                  </ul>

                </div>

              </div>

            </div>


            <div>

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">5.2 Writing Simple Linux Malware in C</h3>

              <p class="text-lg leading-relaxed mb-4">

                Developing simple Linux malware in C, particularly in the form of kernel modules, provides a foundational understanding of how more sophisticated threats operate. <strong>Kernel modules allow code to be loaded and unloaded into the kernel at runtime, granting them kernel-level privileges</strong>.

              </p>


              <div class="code-block mb-6">

                <pre><code>/*

 * Simple Linux kernel module example

 */

#include &lt;linux/init.h&gt;

#include &lt;linux/module.h&gt;

#include &lt;linux/kernel.h&gt;


MODULE_LICENSE(&#34;GPL&#34;);

MODULE_AUTHOR(&#34;EthicalHacker&#34;);

MODULE_DESCRIPTION(&#34;Simple kernel module&#34;);

MODULE_VERSION(&#34;0.01&#34;);


static int __init malicious_init(void) {

    printk(KERN_INFO &#34;Module loaded successfully\n&#34;);

    // Malicious code would go here

    return 0;

}


static void __exit malicious_exit(void) {

    printk(KERN_INFO &#34;Module unloaded\n&#34;);

}


module_init(malicious_init);

module_exit(malicious_exit);</code></pre>

              <p class="text-lg leading-relaxed mb-4">

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">6.2 Linux Privilege Escalation Techniques</h3>

              <p class="text-lg leading-relaxed mb-4">

                <strong>Privilege escalation on Linux systems is a critical step in post-exploitation</strong>, allowing an attacker to gain higher levels of access than initially obtained.

              </p>


              <div class="grid md:grid-cols-2 gap-6 mb-6">

                <div class="bg-blue-50 p-6 rounded-lg border border-blue-200">

                  <h4 class="font-semibold text-lg mb-3 text-blue-900">Horizontal Escalation</h4>

                  <p class="text-sm text-blue-800 mb-3">Gaining access to another user&#39;s resources at the same permission level</p>

                  <ul class="space-y-1 text-xs text-blue-700">

                    <li>• Session hijacking</li>

                    <li>• Credential theft</li>

                    <li>• Token impersonation</li>

                  </ul>

                </div>

                <div class="bg-red-50 p-6 rounded-lg border border-red-200">

                  <h4 class="font-semibold text-lg mb-3 text-red-900">Vertical Escalation</h4>

                  <p class="text-sm text-red-800 mb-3">Elevating from standard user to root/admin privileges</p>

                    <li>• Sudo misconfigurations</li>

                  </ul>

                </div>

              </div>

            <div>

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">8.2 Man-in-the-Middle (MITM) Attacks</h3>

              <p class="text-lg leading-relaxed mb-4">

                <strong>Man-in-the-Middle (MITM) attacks are a type of cyberattack where an attacker secretly intercepts and relays communication between two parties</strong> who believe they are directly communicating with each other.

              </p>


              <div class="bg-yellow-50 p-6 rounded-lg border-l-4 border-yellow-400 mb-6">

                <h4 class="font-semibold text-lg mb-3 text-yellow-900">Common MITM Techniques:</h4>

                <ul class="space-y-2 text-yellow-800">

                  <li><strong>ARP Spoofing/Poisoning</strong> - Redirecting traffic by falsifying ARP messages</li>

                  <li><strong>DNS Spoofing</strong> - Corrupting DNS cache to redirect domain resolutions</li>

                  <li><strong>SSL Stripping</strong> - Downgrading HTTPS connections to HTTP</li>

                  <li><strong>Rogue Access Points</strong> - Creating malicious Wi-Fi hotspots</li>

                </ul>

              </div>


              <div class="code-block">

                <pre><code># ARP spoofing with arpspoof

arpspoof -i eth0 -t 192.168.1.100 192.168.1.1


# MITM with ettercap

ettercap -T -M arp:remote /192.168.1.100/ /192.168.1.1/


# SSL stripping with sslstrip

sslstrip -l 8080</code></pre>

              <h3 class="font-tiempos text-2xl font-bold mb-4 text-gray-800">9.2 Cracking WPA/WPA2 Encryption</h3>

              <p class="text-lg leading-relaxed mb-4">

                <strong>Cracking WPA/WPA2-PSK encryption is a common objective in wireless penetration testing</strong>. The process involves capturing the WPA/WPA2 four-way handshake and performing an offline brute-force or dictionary attack against the network&#39;s password.

                </div>

                  </div>

                  <h4 class="font-semibold text-center mb-2">3. Crack Password</h4>

                </div>

              </div>

        // Initialize Mermaid Controls for zoom and pan

        function initializeMermaidControls() {

            const containers = document.querySelectorAll('.mermaid-container');


            containers.forEach(container => {

            const mermaidElement = container.querySelector('.mermaid');

            let scale = 1;

            let isDragging = false;

            let startX, startY, translateX = 0, translateY = 0;


            // 触摸相关状态

            let isTouch = false;

            let touchStartTime = 0;

            let initialDistance = 0;

            let initialScale = 1;

            let isPinching = false;


            // Zoom controls

            const zoomInBtn = container.querySelector('.zoom-in');

            const zoomOutBtn = container.querySelector('.zoom-out');

            const resetBtn = container.querySelector('.reset-zoom');

            const fullscreenBtn = container.querySelector('.fullscreen');


            function updateTransform() {

                mermaidElement.style.transform = `translate(${translateX}px, ${translateY}px) scale(${scale})`;


                if (scale > 1) {

                container.classList.add('zoomed');

                } else {

                container.classList.remove('zoomed');

                }


                mermaidElement.style.cursor = isDragging ? 'grabbing' : 'grab';

            }


            if (zoomInBtn) {

                zoomInBtn.addEventListener('click', () => {

                scale = Math.min(scale * 1.25, 4);

                updateTransform();

                });

            }


            if (zoomOutBtn) {

                zoomOutBtn.addEventListener('click', () => {

                scale = Math.max(scale / 1.25, 0.3);

                if (scale <= 1) {

                    translateX = 0;

                    translateY = 0;

                }

                updateTransform();

                });

            }


            if (resetBtn) {

                resetBtn.addEventListener('click', () => {

                scale = 1;

                translateX = 0;

                translateY = 0;

                updateTransform();

                });

            }


            if (fullscreenBtn) {

                fullscreenBtn.addEventListener('click', () => {

                if (container.requestFullscreen) {

                    container.requestFullscreen();

                } else if (container.webkitRequestFullscreen) {

                    container.webkitRequestFullscreen();

                } else if (container.msRequestFullscreen) {

                    container.msRequestFullscreen();

                }

                });

            }


            // Mouse Events

            mermaidElement.addEventListener('mousedown', (e) => {

                if (isTouch) return; // 如果是触摸设备，忽略鼠标事件


                isDragging = true;

                startX = e.clientX - translateX;

                startY = e.clientY - translateY;

                mermaidElement.style.cursor = 'grabbing';

                updateTransform();

                e.preventDefault();

            });


            document.addEventListener('mousemove', (e) => {

                if (isDragging && !isTouch) {

                translateX = e.clientX - startX;

                translateY = e.clientY - startY;

                updateTransform();

                }

            });


            document.addEventListener('mouseup', () => {

                if (isDragging && !isTouch) {

                isDragging = false;

                mermaidElement.style.cursor = 'grab';

                updateTransform();

                }

            });


            document.addEventListener('mouseleave', () => {

                if (isDragging && !isTouch) {

                isDragging = false;

                mermaidElement.style.cursor = 'grab';

                updateTransform();

                }

            });


            // 获取两点之间的距离

            function getTouchDistance(touch1, touch2) {

                return Math.hypot(

                touch2.clientX - touch1.clientX,

                touch2.clientY - touch1.clientY

                );

            }


            // Touch Events - 触摸事件处理

            mermaidElement.addEventListener('touchstart', (e) => {

                isTouch = true;

                touchStartTime = Date.now();


                if (e.touches.length === 1) {

                // 单指拖动

                isPinching = false;

                isDragging = true;


                const touch = e.touches[0];

                startX = touch.clientX - translateX;

                startY = touch.clientY - translateY;


                } else if (e.touches.length === 2) {

                // 双指缩放

                isPinching = true;

                isDragging = false;


                const touch1 = e.touches[0];

                const touch2 = e.touches[1];

                initialDistance = getTouchDistance(touch1, touch2);

                initialScale = scale;

                }


                e.preventDefault();

            }, { passive: false });


            mermaidElement.addEventListener('touchmove', (e) => {

                if (e.touches.length === 1 && isDragging && !isPinching) {

                // 单指拖动

                const touch = e.touches[0];

                translateX = touch.clientX - startX;

                translateY = touch.clientY - startY;

                updateTransform();


                } else if (e.touches.length === 2 && isPinching) {

                // 双指缩放

                const touch1 = e.touches[0];

                const touch2 = e.touches[1];

                const currentDistance = getTouchDistance(touch1, touch2);


                if (initialDistance > 0) {

                    const newScale = Math.min(Math.max(

                    initialScale * (currentDistance / initialDistance),

                    0.3

                    ), 4);

                    scale = newScale;

                    updateTransform();

                }

                }


                e.preventDefault();

            }, { passive: false });


            mermaidElement.addEventListener('touchend', (e) => {

                // 重置状态

                if (e.touches.length === 0) {

                isDragging = false;

                isPinching = false;

                initialDistance = 0;


                // 延迟重置isTouch，避免鼠标事件立即触发

                setTimeout(() => {

                    isTouch = false;

                }, 100);

                } else if (e.touches.length === 1 && isPinching) {

                // 从双指变为单指，切换为拖动模式

                isPinching = false;

                isDragging = true;


                const touch = e.touches[0];

                startX = touch.clientX - translateX;

                startY = touch.clientY - translateY;

                }


                updateTransform();

            });


            mermaidElement.addEventListener('touchcancel', (e) => {

                isDragging = false;

                isPinching = false;

                initialDistance = 0;


                setTimeout(() => {

                isTouch = false;

                }, 100);


                updateTransform();

            });


            // Enhanced wheel zoom with better center point handling

            container.addEventListener('wheel', (e) => {

                e.preventDefault();

                const rect = container.getBoundingClientRect();

                const centerX = rect.width / 2;

                const centerY = rect.height / 2;


                const delta = e.deltaY > 0 ? 0.9 : 1.1;

                const newScale = Math.min(Math.max(scale * delta, 0.3), 4);


                // Adjust translation to zoom towards center

                if (newScale !== scale) {

                const scaleDiff = newScale / scale;

                translateX = translateX * scaleDiff;

                translateY = translateY * scaleDiff;

                scale = newScale;


                if (scale <= 1) {

                    translateX = 0;

                    translateY = 0;

                }


                updateTransform();

                }

            });


            // Initialize display

            updateTransform();

            });

        }


        // Initialize Mermaid Controls

        initializeMermaidControls();

        const mobileMenuToggle = document.getElementById('mobile-menu-toggle');

        const toc = document.getElementById('toc');

        const tocOverlay = document.getElementById('toc-overlay');

        

        mobileMenuToggle.addEventListener('click', () => {

            toc.classList.toggle('mobile-open');

            tocOverlay.classList.toggle('active');

        });


        // Close mobile menu when clicking overlay

        tocOverlay.addEventListener('click', () => {

            toc.classList.remove('mobile-open');

            tocOverlay.classList.remove('active');

        });

        document.querySelectorAll('.toc-link').forEach(link => {

            link.addEventListener('click', (e) => {

                e.preventDefault();

                const targetId = link.getAttribute('href').substring(1);

                const targetElement = document.getElementById(targetId);

                

                if (targetElement) {

                    targetElement.scrollIntoView({

                        behavior: 'smooth',

                        block: 'start'

                    });

                }

                

                // Close mobile menu

                toc.classList.remove('mobile-open');

                tocOverlay.classList.remove('active');

            });

        });

        const sections = document.querySelectorAll('section[id]');

        const tocLinks = document.querySelectorAll('.toc-link');


        function updateActiveSection() {

            let current = '';

            sections.forEach(section => {

                const sectionTop = section.offsetTop;

                // const sectionHeight = section.clientHeight; // Not used in this logic

                if (window.pageYOffset >= sectionTop - 100) { // Adjust offset as needed

                    current = section.getAttribute('id');

                }

            });


            tocLinks.forEach(link => {

                link.classList.remove('active');

                if (link.getAttribute('href') === '#' + current) {

                    link.classList.add('active');

                }

            });

        }


        window.addEventListener('scroll', updateActiveSection);

        updateActiveSection(); // Initial call
